## 4.8 まとめ

* `F#`と`Go`の型システムについて

| 項目   | `F#`   | Go           |
|------|--------|--------------|
| 型の合成 | AND型   | 構造体          |
|      | OR型    | インターフェース／列挙型 |
| 値がない | Option | ポインタ型(`*T`)  |
| エラー   | Result | タプル(`(T, error)`)  |
|      |        | インターフェース(`error`) |
| コレクション | `list`, `seq`, `array` | `slice`, `map` |
